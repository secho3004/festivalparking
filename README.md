5. **결측치 처리** : 요금 관련 컬럼 결측률 70% 이상으로 분석에서 제외. 위도/경도 결측은 유지

## (3) 시각화 결과

### 차트 1. 시도별 축제 수 vs 총 주차구획수
**SQL**
```sql
SELECT 시도, 축제수, 총주차구획수
FROM region_summary
ORDER BY 축제수 DESC;
```
**인사이트**  
경기·서울 등 수도권은 축제 수와 주차 구획수 모두 높은 반면, 제주·강원은 축제 수 대비 주차 구획수가 상대적으로 부족하여 성수기 교통 혼잡 위험이 높음을 확인할 수 있다.

---

### 차트 2. 월별 축제 개최 현황
**SQL**
```sql
SELECT 월, COUNT(*) AS 축제수, AVG(기간_일수) AS 평균기간
FROM festival
WHERE 월 IS NOT NULL
GROUP BY 월
ORDER BY 월;
```
**인사이트**  
봄(4~5월)과 가을(9~10월)에 축제가 집중되는 뚜렷한 계절성 패턴이 나타난다. 특히 10월은 축제 수가 가장 많아 해당 시기 주차 수요가 급증할 가능성이 높다.

---

### 차트 3. 시도별 공영 vs 민영 주차장 비율
**SQL**
```sql
SELECT 시도, 주차장구분, COUNT(*) AS 개수
FROM parking
WHERE 시도 IS NOT NULL
GROUP BY 시도, 주차장구분;
```
**인사이트**  
전국 대부분 지역에서 공영 주차장 비중이 압도적으로 높다. 민영 주차장 비율이 낮은 지역은 축제 기간 민간 협력 주차 공간 확보가 어려워 공영 주차장 의존도가 높아질 수 있다.

---

### 차트 4. 축제 수 대비 주차구획수 산점도
**SQL**
```sql
SELECT 시도, 축제수,
       ROUND(CAST(총주차구획수 AS FLOAT) / 축제수, 1) AS 축제당_주차구획수
FROM region_summary
WHERE 축제수 > 0;
```
**인사이트**  
축제 수가 많으면서 축제당 주차구획수가 적은 지역(우측 하단)이 인프라 부족 위험 지역이다. 반대로 축제는 적지만 주차 여건이 충분한 지역(좌상단)은 상대적으로 여유로운 환경을 갖추고 있다.
